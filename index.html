The Verification Horizon: No silver bullet for coding agent rewards

Hallucination in world models is predictable and preventable

UC San Diego researchers trained a 350M-parameter world model on 210 tasks and showed hallucination is primarily a data-coverage problem, not architectural — predictable from runtime uncertainty signals and preventable through targeted training data rather than model redesign.

Why multi-step tool-use reinforcement learning collapses

Structural collapse — not reasoning loss — is the core failure mode when RL is applied to multi-step tool use in LLMs. Interleaving SFT with RL, especially Process Reflection Supervision, significantly improves stability and generalization to out-of-distribution scenarios.

GUI vs. CLI: Execution bottlenecks in computer-use agents

Matched benchmark of 440 desktop tasks across 18 applications shows strongest GUI agent (GPT-5.4) at 59.1% vs. 48.2% for CLI — but verifier-guided CLI augmentation raises CLI to 69.3%, proving the gap stems from incomplete skill coverage, not model capability alone.

OPID: On-policy skill distillation for agentic reinforcement learning

Extracts hierarchical hindsight skills from completed RL trajectories to provide dense token-level supervision for LLM agents. A critical-first routing mechanism injects skill context at pivotal timesteps, yielding measurable gains over outcome-only RL on ALFWorld, Search-based QA, and WebShop.
